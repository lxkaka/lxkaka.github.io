lxkaka

ж¶ҲжҒҜзі»з»ҹж–№жЎҲи®ҫи®Ўд№Ӣз«ҷеҶ…ж¶ҲжҒҜе’ҢдәӢд»¶жҸҗйҶ’

lxkaka еҸ‘еёғдәҺ 2022-04-22 ж”¶еҪ•дәҺ service backend е®һзҺ°
ж¶ҲжҒҜзі»з»ҹеҸҜд»ҘиҜҙжҳҜжҜҸдёӘеә”з”ЁдёҚеҸҜзјәе°‘зҡ„зі»з»ҹгҖӮд»ҺеҠҹиғҪдёҠжқҘиҜҙдёҖдёӘж¶ҲжҒҜзі»з»ҹиҮіе°‘жңүдёӨеӨ§йғЁеҲҶз»„жҲҗ ж¶ҲжҒҜзҡ„еӯҳеӮЁ ж¶ҲжҒҜзҡ„йҖҡзҹҘ ж¶ҲжҒҜзі»з»ҹзҡ„еӨҚжқӮзЁӢеәҰе’Ңз”ЁжҲ·и§„жЁЎе’ҢдёҡеҠЎи§„жЁЎ
йҳ…иҜ»е…Ёж–Ү
 ж¶ҲжҒҜ, ж–№жЎҲи®ҫи®Ў, mysql

еҲ©з”Ё Casbin й«ҳж•Ҳе®һзҺ°еҹәдәҺ RBAC зҡ„жқғйҷҗзі»з»ҹ

lxkaka еҸ‘еёғдәҺ 2022-03-25 ж”¶еҪ•дәҺ service golang е®һзҺ°
еңЁеӨ§йғЁеҲҶзҡ„дёҡеҠЎзі»з»ҹдёӯжқғйҷҗжҺ§еҲ¶жҳҜе°‘дёҚдәҶзҡ„жЁЎеқ—пјҢе®ғеҜ№иҝҷдёӘзі»з»ҹзҡ„ж•°жҚ®е®үе…Ёиө·зқҖе…ій”®дҪңз”ЁгҖӮж—ўз„¶жқғйҷҗзі»з»ҹжҲ–иҖ…жЁЎеқ—иҝҷд№ҲйҮҚиҰҒдё”еҸҲеҫҲжҷ®йҖӮпјҢеҪ“жҲ‘д»¬жҗӯе»әдёҖдёӘжқғйҷҗзі»
йҳ…иҜ»е…Ёж–Ү
 жқғйҷҗзі»з»ҹ, RBAC, Casbin

з®—жі•жңҚеҠЎзЁіе®ҡжҖ§дјҳеҢ–д№ӢиҙҹиҪҪжҺ§еҲ¶

lxkaka еҸ‘еёғдәҺ 2022-02-27 ж”¶еҪ•дәҺ service grpc backend
еңЁжҲ‘д»¬зҡ„з®—жі•жңҚеҠЎдёӯпјҢжҲ‘д»¬йҖҡиҝҮ gRPC server еҜ№еӨ–жҸҗдҫӣжңҚеҠЎгҖӮиҝҷйҮҢз®—жі•е’ҢдёҖиҲ¬зҡ„жңҚеҠЎжңүдёҖдёӘеҫҲеӨ§зҡ„еҢәеҲ«, з®—жі•жңҚеҠЎдҫқиө–дәҺ GPU, еҫҖеҫҖдёҖж¬ЎиҜ·жұӮе°ұиғҪжҠҠеҚ•еҚЎзҡ„жҖ§иғҪеҗғеҲ° 80%гҖӮ
йҳ…иҜ»е…Ёж–Ү
 иҙҹиҪҪеқҮиЎЎ, жңҚеҠЎжІ»зҗҶ, P2C

жҳ“дәҺдҪҝз”Ёзҡ„жҖ§иғҪжөӢиҜ•е·Ҙе…·-Locust

lxkaka еҸ‘еёғдәҺ 2022-01-29 ж”¶еҪ•дәҺ service backend е·ҘзЁӢж•ҲзҺҮ
еҪ“жҲ‘д»¬еҜ№еӨ–жҸҗдҫӣжңҚеҠЎзҡ„ж—¶еҖҷи°ғз”Ёж–№з»Ҹеёёдјҡй—®иҝҷж ·дёҖдёӘй—®йўҳпјҡдҪ д»¬зҡ„жңҚеҠЎиғҪжҸҗдҫӣеӨҡеӨ§зҡ„ QPSпјҹ йӮЈд№Ҳе°ұиҰҒжұӮжҲ‘д»¬еңЁжңҚеҠЎдёҠзәҝд№ӢеүҚе°ұеә”иҜҘжё…жҘҡзҡ„зҹҘйҒ“жңҚеҠЎзҡ„жһҒйҷҗ QPS жҳҜ
йҳ…иҜ»е…Ёж–Ү
 еҺӢжөӢ, жҖ§иғҪ

з”Ё StatefulSet е®һзҺ°еҗҢжӯҘжңҚеҠЎе’ҢејӮжӯҘжңҚеҠЎзҡ„з®ЎзҗҶ

lxkaka еҸ‘еёғдәҺ 2021-12-26 ж”¶еҪ•дәҺ backend infra
дёҖдёӘжңҚеҠЎйҮҢж—ўжңүеҗҢжӯҘйҖ»иҫ‘еҸҲжңүејӮжӯҘйҖ»иҫ‘жҳҜйқһеёёеёёи§Ғзҡ„дәӢпјҢжҜ”еҰӮеҸҜд»ҘйҖҡиҝҮ Http жҺҘеҸЈи°ғз”ЁжңҚеҠЎжҲ–иҖ…йҖҡиҝҮж¶ҲжҒҜйҳҹеҲ—дј йҖ’ж¶ҲжҒҜжқҘе®һзҺ°еҗҢж ·зҡ„жңҚеҠЎйҖ»иҫ‘пјҢеҗҢдёҖеҘ—д»Јз ҒйҖ»иҫ‘еҢәеҲ«
йҳ…иҜ»е…Ёж–Ү
 жңҚеҠЎжІ»зҗҶ, k8s

д»ҺдёҖдёӘж¶Ҳиҙ№ж…ўзҡ„дҫӢеӯҗж·ұе…ҘзҗҶи§Ј kafka rebalance

lxkaka еҸ‘еёғдәҺ 2021-11-20 ж”¶еҪ•дәҺ еҺҹзҗҶ backend infra
ж¶ҲжҒҜйҳҹеҲ—жҳҜжңҚеҠЎз«Ҝеҝ…дёҚеҸҜе°‘зҡ„з»„д»¶пјҢе…¶дёӯ Kafka еҸҜд»ҘиҜҙжҳҜж•°дёҖж•°дәҢзҡ„йҖүжӢ©пјҢеҜ№дәҺеӨ§йғЁеҲҶжңҚеҠЎз«Ҝзҡ„еҗҢеӯҰжқҘиҜҙ Kafka д№ҹжҳҜжңҖзҶҹжӮүзҡ„ж¶ҲжҒҜдёӯй—ҙд»¶д№ӢдёҖгҖӮиҖҢеҪ“жҲ‘д»¬еңЁз”ҹдә§дёҠйҒҮеҲ° kafka
йҳ…иҜ»е…Ёж–Ү
